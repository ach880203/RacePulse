- 쌍승식(EXA) : 1등과 2등으로 들어올 말 2두를 순서대로 적중시키는 방식입니다.
- 삼복승식(TLA) : 1등, 2등 및 3등으로 들어올 말 3두를 순서에 상관없이 적중시키는 방식입니다.
- 삼쌍승식(TRI) : 1등, 2등, 3등으로 들어올 말 3두를 순서대로 적중시키는 방식입니다.
